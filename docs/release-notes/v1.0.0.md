BandBuddy `1.0.0` 是项目的第一个公开版本：一个围绕“把难点真正练会”设计的本地优先 Windows 练琴工作台。

## 这次可以做什么

- 在本机把歌曲分离为 Vocal、Drums、Bass、Guitar、Piano、Other 六轨，或直接导入已有分轨。
- 在同步波形上独立 Mute、Solo、调节增益，使用 0.20×–4.00× 保持音高的变速和 A–B 循环拆练难点。
- 自动检测 BPM，使用拍点微调、节拍器与 4/8 拍预备拍练准进入时机。
- 自动保存播放位置、循环、速度和混音状态，下次从同一练习现场继续。
- 导出 WAV、FLAC、MP3 标准分轨，或导出应用当前速度和 A–B 范围的练习混音。
- 用曲库管理收藏、最近练习和后台任务；整个音频处理流程无需上传音乐。

## 下载

- `BandBuddy-1.0.0-x64.exe`：Windows 10/11 x64 安装版。
- `BandBuddy-1.0.0-x64-portable.exe`：Windows 10/11 x64 便携版。
- `SHA256SUMS.txt`：两份可执行文件的 SHA-256 校验值。

首次使用 AI 分轨前，请在设置中安装本地运行环境，并预留约 8–15 GB 空间。项目不附带音乐；请只处理你拥有或获授权使用的音频。
